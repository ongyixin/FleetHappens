Getting Started

  • GOOGLE_GEM_USER_GUIDE.md - Easiest path! Describe (or draw) what you want, paste JSON, done. No coding required.
  • INSTANT_START_WITH_CLAUDE.md - Zero to working code in 60 seconds using Claude. Includes network egress setup.
  • BEGINNER_GUIDE.md - New to coding? Start here for plain-English explanations of APIs, terminals, and more.
  • CREDENTIALS.md - How to set up your .env file with Geotab API credentials.
  • CLAUDE_PROMPTS.md - Copy-paste prompts for common Geotab tasks.
