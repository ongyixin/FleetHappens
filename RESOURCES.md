## üêõ Debugging & Troubleshooting

### Common Geotab API Errors

**Error: "Incorrect login credentials"**
- Check your database name (case sensitive!)
- Verify username and password
- Ensure you're using the correct server URL
- Demo accounts: server is usually `my.geotab.com`

**Error: "Rate limit exceeded"**
- Implement request throttling
- Cache responses when possible
- Use search parameters to limit data returned
- See: https://geotab.github.io/sdk/software/guides/concepts/#api-rate-limits

**Error: "Invalid method"**
- Check API reference for correct method name
- Verify you're using the right SDK version
- Ensure method exists for your API version

**Error: "Missing required parameter"**
- Review API reference for required fields
- Check parameter spelling and case
- Ensure credentials object is complete

### Debugging Tools
- **Chrome DevTools**: https://developer.chrome.com/docs/devtools/
- **Firefox DevTools**: https://firefox-source-docs.mozilla.org/devtools-user/
- **Node.js Debugger**: https://nodejs.org/en/docs/guides/debugging-getting-started/
- **Python Debugger (pdb)**: https://docs.python.org/3/library/pdb.html

### Stack Overflow
Search for: `[geotab]` + your error message
- Example: `[geotab] authentication failed`
